- **간트차트 보기**: `index.html` 또는 `국민통합위_3건_간트차트.html` 을 브라우저로 열면 됩니다.
- 좌측: 프로젝트/행사 · 우측: 2026년 6~10월 타임라인
- 색상: 세대젠더(파랑) · 경제양극화(초록) · 경청소통(주황)
- 확정 일정과 가안(빗금 표시) 구분, 오늘 날짜 표시(빨간 선)

## 편집 방법
협업자로 초대된 분은 파일을 직접 수정 후 커밋/푸시하거나, 웹에서 파일을 열어 ✏️ 아이콘으로 편집할 수 있습니다.

## Slack에서 Codex로 수정 요청

Slack 채널에 `@Codex`를 초대한 뒤, 아래처럼 요청하면 이 저장소의 현황판 수정 작업을 시작할 수 있습니다.

```text
@Codex syoug93/k-cohesion 저장소의 일정 현황판을 수정해줘.
이 스레드의 수정 의견을 반영하고, index.html과 국민통합위_3건_간트차트.html을 동일하게 업데이트해줘.
완료 후 변경 요약과 public 링크를 알려줘.
```

저장소 작업 규칙은 `AGENTS.md`에 정리되어 있습니다.

## Google Calendar 자동화 방향

일정을 Google Calendar에 먼저 등록하는 운영이라면, 별도 프로젝트 캘린더 또는 비공개 iCal 피드를 기준으로 GitHub Actions가 주기적으로 현황판을 갱신하는 구조로 자동화할 수 있습니다.

권장 방식:

- 캘린더 이벤트 제목에 `[세대젠더]`, `[경제양극화]`, `[경청소통]` 같은 프로젝트 표식을 붙입니다.
- GitHub Secret에 Google Calendar iCal URL을 저장합니다.
- 동기화 스크립트가 캘린더 이벤트를 읽어 HTML 현황판을 갱신합니다.
- 초기에는 자동 커밋보다 PR 생성 방식으로 운영해 검토 후 반영하는 것을 권장합니다.
